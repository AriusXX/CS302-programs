#Purpose

This program simulates the Shamrock Race, where contestants compete in different race types. Players can enter one of three events:

One-Legged Race – A test of balance and endurance
Sprint Race – A fast-paced competition of speed
Bicycle Race – A race of skill and strategy on two wheels


#How to use
The program provides a menu-driven system that allows users to interact with a Binary Search Tree (BST). By selecting a menu option with an integer input, users can:

1.Add a new contestant to the race
2. Display all registered contestants
3. Remove a contestant from the tree
4. Clear all contestants from the tree
5. Exit the program


#Creating a contestant
When adding a contestant, users will:

1. Enter the contestant’s name
2. Select a race type from the menu
3. Provide additional details (if needed, such as speed)
4. Each race type has unique characteristics, making every contestant different from the others.
